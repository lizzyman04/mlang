MLangA Math-First Programming Language

Statically typed, expressive, and built in Rust. Write clean scripts with native structs, functions, and I/O — no boilerplate.

Get Started
View on GitHub
🔢

Statically Typed

Every variable has an explicit type — int, dec, txt, bool, array, or struct. No runtime surprises.

⚡

Fast & Lightweight

Lexer → Parser → AST → Evaluator pipeline with zero external runtime dependencies. Starts instantly.

🏗️

Structs & Functions

First-class user-defined structs with field access and mutation. Functions support recursion and struct-typed parameters.

📥

Unified I/O

read(), read_int(), read_dec() with optional inline prompts. Type conversion via int(), dec(), txt() casts.

🔁

Rich Control Flow

if/else, while loops, for-range loops, for-in array loops, break, continue, and early return.

📦

Array Builtins

push, pop, clear, len, contains, slice — all as ergonomic method calls directly on array variables.

A taste of MLang ​

mlang
struct Point {
    int x
    int y
}

int distance_sq(Point a, Point b) {
    int dx = a.x - b.x;
    int dy = a.y - b.y;
    return dx * dx + dy * dy;
}

main() {
    let p1 = Point { x = 0, y = 0 };
    let p2 = Point { x = 3, y = 4 };
    print(distance_sq(p1, p2));   # 25

    txt name = read("Your name: ");
    print("Hello, " + name);
}
